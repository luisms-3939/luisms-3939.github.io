Completed the **Google Project Management Professional Certificate**, a 6-course program covering traditional and Agile project management methodologies, preparing learners for entry-level project management roles.

## Skills Gained

Developed expertise in **project planning**, **risk management**, **stakeholder communication**, **Agile methodologies**, and using project management tools to deliver projects on time and within budget.

## Course Curriculum

- Foundations of Project Management
- Project Initiation: Starting a Successful Project
- Project Planning: Putting It All Together
- Project Execution: Running the Project
- Agile Project Management
- Capstone: Applying Project Management in the Real World

## Key Takeaways

This certification provided practical experience with both Waterfall and Agile frameworks, including Scrum and Kanban methodologies. Learned to create project charters, manage timelines using Gantt charts, facilitate sprint planning, and navigate common project challenges while maintaining stakeholder alignment.
